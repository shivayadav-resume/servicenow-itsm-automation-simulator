# ServiceNow ITSM Automation Simulator

This project simulates ServiceNow ITSM incident lifecycle logic using JavaScript.

## What This Project Demonstrates
- Incident priority calculation (Business Rules)
- Auto assignment logic (Assignment Rules)
- SLA breach detection
- Incident state transitions

## Why This Project Exists
Due to restricted access to client ServiceNow instances, this project demonstrates
hands-on understanding of ServiceNow ITSM processes using JavaScript logic similar
to Business Rules and Script Includes.

## Technologies Used
- JavaScript
- JSON
